# ①課題名
掲示板アプリ（PHP版）

## ②課題内容（どんな作品か）
-スレッドを立てられるようにしたPHPバージョンの掲示板アプリです

## ⑤工夫した点・こだわった点
- 今回授業で勉強した内容のアプリを手で作った後、今までに作った内容をPHPで再現してみました
- 今までは授業で習った範囲をどう身に着けるかに四苦八苦していたのですが、いろいろできることが
  増えてきていて、もっと拡大していろいろなことに手を出さないといけないなと気づきました
- いままでやってきたことが少しずつつながり始めていて、楽しいです
## ⑥難しかった点・次回トライしたいこと（又は機能）
- 掲示板形式以外に挑戦したい。また、今までは機能の一部だけだったので、
  ＰＨＰを活用してサイトの構成をどんどん考えていきたい。

## ⑦フリー項目（感想、シェアしたいこと等なんでも）
- 自分のやりたいことに、掲示板という形があっていない気がしてきました。
- 完全に言葉を自分で作ろうとすると、少し気恥ずかしさや心理的なハードルが上がる気がするので、
  boketeのような大喜利形式で、概念、読み方、当て字のどれかを募集するような形式
- それぞれの大喜利のお題にコメントがつけられて、知恵袋やユーチューブのように人気の回答が上位に来る
- リアクションスタンプが豊富だと楽しいかななどいろいろアイディアがわいてきています
